## 📝 Descrição
Este projeto foi desenvolvido com o objetivo de aplicar os conceitos de **pensamento computacional** na concepção de um sistema de larga escala. O foco é explorar a decomposição de problemas complexos, abstração de dados, reconhecimento de padrões e a criação de algoritmos eficientes.

## 🎯 Objetivos
*   Relacionar Engenharia de Software e Pensamento Computacional;
*   Reconhecer princípios e padrões para sistemas de larga escala;
*   Identificar dificuldades reais no desenvolvimento de aplicações complexas;
*   Aplicar metodologias ágeis no planejamento.

---

## 🏛️ Sistema Proposto: Plataforma Acadêmica Inteligente

Uma aplicação web para gestão acadêmica que integra:
1. **Cadastro e Autenticação:** Segurança e controle de acesso.
2. **Módulo de Disciplinas e Notas:** Organização curricular.
3. **Sistema de Recomendações (IA):** Sugestões personalizadas para alunos.
4. **Painel de Relatórios:** Visão analítica para a coordenação.

### 🧠 Pensamento Computacional Aplicado

| Pilar | Aplicação no Projeto |
| :--- | :--- |
| **Decomposição** | Divisão em módulos: Autenticação, Notas, Relatórios e IA. |
| **Reconhecimento de Padrões** | Login baseado em segurança bancária; estrutura de notas padrão LMS. |
| **Abstração** | Uso de diagramas UML para simplificar a visão dos módulos. |
| **Algoritmos** | Lógica de cálculo de médias e fluxos de decisão de IA. |
